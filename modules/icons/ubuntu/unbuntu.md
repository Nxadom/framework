# 🎨 Daftar Icon Ubuntu Yaru

Semua icon ini berasal dari folder `assets/modules/icons/ubuntu/` dan didefinisikan di `ubuntu.css`.

---

## 📦 Cara Pakai

### Monochrome (symbolic) — `uicon`
Warna mengikuti `currentColor` — bisa diubah via CSS `color`.

```html
<span class="uicon uicon-actions-16-close"></span>
<span class="uicon uicon-status-symbolic-battery-full-symbolic" style="color:green"></span>
```

### Colored (full color) — `uicon-img`
Menampilkan warna asli SVG.

```html
<span class="uicon-img uicon-img-apps-16-firefox"></span>
```

### Ukuran
Tambah class `uicon:16`, `uicon:24`, `uicon:32`, `uicon:48`, atau `uicon:64`.

```html
<span class="uicon uicon:24 uicon-actions-16-close"></span>
```

## 🏗️ Cara mencari icon

Gunakan format nama untuk menebak:

```
.uicon-{kategori}-{ukuran}-{nama-icon}
.uicon-img-{kategori}-{ukuran}-{nama-icon}
```

Contoh pencarian mental:
- Mau icon **close**? → `.uicon-actions-16-close`
- Mau icon **folder**? → `.uicon-img-places-16-folder`
- Mau icon **baterai**? → `.uicon-status-16-battery-full`
- Mau icon **Firefox**? → `.uicon-img-apps-16-firefox`
- Mau icon **volume**? → `.uicon-status-symbolic-audio-volume-high-symbolic`
- Mau icon **Download**? → `.uicon-img-places-16-folder-download` atau `.uicon-actions-16-browser-download`
