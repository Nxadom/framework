# NexaInteract.js - Fitur dan Dokumentasi Lengkap

## Ringkasan Umum

NexaInteract.js adalah library JavaScript yang menyediakan sistem context menu interaktif untuk manipulasi elemen HTML secara real-time. Library ini memungkinkan pengguna untuk mengedit, menyalin, memotong, dan memasukkan berbagai elemen HTML melalui context menu yang dapat dikustomisasi.

## Fitur Utama

- **Context Menu yang Dapat Dikustomisasi**: Menu klik kanan dengan berbagai opsi editing
- **Submenu Bertingkat**: Dukungan untuk submenu dengan kategori yang terorganisir
- **Positioning Otomatis**: Menu secara otomatis menyesuaikan posisi untuk tetap dalam viewport
- **Event Handling**: Penanganan event mouse, keyboard, dan sentuhan
- **Print Mode Detection**: Deteksi mode print dan penyembunyian menu otomatis

## Penggunaan

```javascript
// Inisialisasi dasar
const nexaInteract = new NexaInteract();

// Konfigurasi lanjutan
const nexaInteract = new NexaInteract({
  enabledIds: ["content", "editor"],
  enabledClasses: ["editable", "interactive"],
  customMenuItems: [
    {
      text: "Custom Action",
      action: "customAction",
      icon: "star",
    },
  ],
  unsplashAccessKey: "your-api-key",
});
```
