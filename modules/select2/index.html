Select2

Select2 dengan tema Primer CSS — single, multiple, searchable, grouping, ukuran, dan state.

1. Single — Default

Gunakan .select2() tanpa opsi tambahan.

2. Single — Placeholder

Tambahkan placeholder dan <option value=""> kosong di awal.

3. Single — Allow Clear (×)

Tombol × muncul saat nilai dipilih. Butuh allowClear: true + placeholder.

4. Single — Disabled

Tambahkan atribut disabled pada elemen <select>.

5. Single — Tanpa Kotak Pencarian

Gunakan minimumResultsForSearch: Infinity untuk menyembunyikan input pencarian.

6. Multiple — Pilih Banyak

Tambahkan atribut multiple pada <select>.

7. Multiple — Maks. 3 Pilihan

Gunakan maximumSelectionLength: 3 — pilihan ke-4 akan ditolak.

8. Tags — Input Bebas

Aktifkan tags: true — pengguna dapat mengetik nilai baru yang tidak ada di daftar.

9. Option Groups

Gunakan <optgroup> untuk mengelompokkan pilihan.

10. Option Disabled

Tambahkan disabled pada <option> yang tidak bisa dipilih.

11. Ukuran — Small

Tambahkan class select2-container--small via containerCssClass.

12. Ukuran — Extra Small

← ukuran xs cocok untuk toolbar / filter

13. Dalam .form-group

Select2 berjalan di dalam .form-group standar.

Role wajib dipilih.
