Alerts / Flash

Komponen flash alert dengan tema Primer CSS — info, warning, error, success, dismissible, icon, dan banner.

1. Variants

Ini adalah pesan informasi — default flash (biru).
Ini adalah pesan peringatan — harap perhatikan sebelum melanjutkan.
Ini adalah pesan kesalahan — tindakan tidak dapat diselesaikan.
Ini adalah pesan berhasil — perubahan telah disimpan.

Class: .flash, .flash-warn, .flash-error, .flash-success

2. Dismissible — Bisa Ditutup

Pembaruan tersedia. Pelajari lebih lanjut tentang fitur baru.
Sesi Anda akan berakhir dalam 5 menit. Simpan pekerjaan Anda sekarang.
Gagal menghubungi server. Periksa koneksi internet Anda.
Profil berhasil diperbarui!

Tambahkan .flash-dismiss button di dalam .flash.

3. Dengan Icon

Pembaruan tersedia untuk repositori Anda. Lihat changelog
Akses ke fitur ini akan berakhir pada 1 Mei 2026. Upgrade sekarang
Autentikasi gagal. Token Anda tidak valid atau sudah kadaluarsa.
Deploy berhasil. Versi v2.4.1 sudah live di production.

Gunakan .flash-with-icon + .flash-icon + .flash-content.

4. Dengan Judul

Pembaruan Tersedia

Versi terbaru dari framework ini sudah tersedia. Lakukan pembaruan untuk mendapatkan fitur baru dan perbaikan keamanan.

Terjadi Kesalahan

Permintaan Anda tidak dapat diproses karena data tidak lengkap. Periksa kembali semua field yang wajib diisi.

Tambahkan <p class="flash-title"> di dalam .flash-content.

5. Dengan Tombol Aksi

Repositori ini belum memiliki lisensi. Tambahkan lisensi agar orang lain dapat menggunakan kode Anda.

Gunakan .flash-actions di dalam .flash-content.
